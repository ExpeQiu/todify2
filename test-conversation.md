# 多轮对话功能测试指南

## 测试步骤

### 1. 访问AI问答页面
- 打开浏览器访问 http://localhost:3001/
- 点击左侧导航栏的"AI问答"

### 2. 测试单轮对话
- 在输入框中输入问题："什么是人工智能？"
- 点击发送按钮
- 观察AI回复是否正常显示

### 3. 测试多轮对话
- 在AI回复后，继续提问："它有哪些应用领域？"
- 观察页面顶部是否显示"多轮对话中"状态
- 检查AI是否能理解上下文并给出相关回答

### 4. 测试新对话功能
- 点击"新对话"按钮
- 观察"多轮对话中"状态是否消失
- 输入新问题验证对话已重置

### 5. 测试Dify API集成
- 检查浏览器开发者工具的Network标签
- 观察API请求是否包含conversation_id参数
- 验证多轮对话时conversation_id是否保持一致

## 预期结果

✅ AI能正常回复问题
✅ 多轮对话时显示状态指示器
✅ 新对话按钮能重置对话状态
✅ API请求正确传递conversation_id
✅ 界面响应流畅，无错误提示

- [ ] 多轮对话上下文连贯
